[Groovy] Class StudioTechWebDriverEventListener

    • com.disney.studio.qa.stbx.eventlistener.StudioTechWebDriverEventListener
    • All Implemented Interfaces and Traits:
      org.openqa.selenium.support.events.WebDriverEventListener

      @groovy.util.logging.Slf4j
class StudioTechWebDriverEventListener
extends java.lang.Object

      This class is responsible for trapping WebDriver events. These events include, but are not limited to,

      • beforeNavigateTo
      • beforeFindBy
      • afterFindBy
      Not all events within this class have an implementation. For those that do, the implementation logs various aspects of the event to a log file contained within the project's target/logs folder.
